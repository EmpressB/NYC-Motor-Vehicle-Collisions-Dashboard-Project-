


# NYC Motor Vehicle Collisions Dashboard

## Project Overview
This project analyzes motor vehicle collision trends in New York City, focusing on temporal patterns, borough-level differences, and contributing factors to crashes and injuries.

## Tools Used
- Tableau Public
- Python (data cleaning)
- NYC Open Data

## Key Focus Areas
- Yearly and monthly crash trends
- High-risk times and locations
- Pedestrian injury distribution by borough
- Contributing factors to collisions
